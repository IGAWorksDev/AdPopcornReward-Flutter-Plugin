# adpopcornreward

AdPopcornReward plugin project.

## Current AdPopcornReward SDK version

- Android : 9.2.0
- iOS : 5.2.0

## Install the Plugin

Add dependencies on pubspec.yaml

```
dependencies:
  adpopcornreward: ^1.0.4
```
